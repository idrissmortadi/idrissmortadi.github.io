Drawing the Mandelbrot set with C !

In this project, I developed a program in C to generate and visualize the Mandelbrot set, a famous fractal pattern known for its intricate and infinitely complex structure.

The Mandelbrot set is defined by iterating the equation \(z_{n+1}=z_n^2 + C\), where \(z_n\) and \(C\) are complex numbers. The set itself consists of all values of \(C\) for which the sequence does not tend to infinity.
